LOVE EFFECT - une petite surprise (1 minute, puis retour a la normale)

CE QUE CA FAIT
- Affiche des coeurs et le message "Sorry but I love u"
- Change temporairement le fond d'ecran et les icones des dossiers du bureau
- Remplit le bureau de dossiers "KNBGHIK" avec une icone coeur
- Au bout d'1 minute : TOUT est restaure automatiquement

CE QUE CA NE FAIT PAS
- Aucun droit administrateur demande
- Rien d'installe en permanence, rien de cache
- Une fenetre demande votre accord avant de demarrer

POUR LANCER
1. Double-cliquez sur "Lancez-moi.bat"
2. Une fenetre s'ouvre : cliquez "Oui" pour lancer, "Non" pour annuler

SI QUELQUE CHOSE RESTE APRES (ex: PC eteint pendant l'effet)
Ouvrez PowerShell dans ce dossier et lancez :
    python love_mode.py --restore

A PROPOS DES AVERTISSEMENTS WINDOWS (si vous utilisez l'exe)
Un .exe non signe declenche "Windows a protege votre PC" (SmartScreen).
C'est normal pour tout petit programme sans reputation. Le chemin honnete :
cliquer "Informations complementaires" -> "Executer quand meme".
Il n'existe pas d'astuce gratuite pour faire disparaitre cet avertissement
sans un vrai certificat de signature de code (payant, avec reputation).
